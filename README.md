# 📎 PDF Merger - Larga Escala

Executável para realizar a **mesclagem de PDFs em larga escala**.

A ferramenta é uma alternativa para situações em que plataformas online, como iLovePDF e similares, apresentem limitações relacionadas à quantidade de arquivos ou ao tamanho total dos documentos.

---

## 🚀 Como utilizar

1. Copie o arquivo `junta_pdf.exe`
2. Cole na pasta onde estão os PDFs que deseja unir
3. Execute com duplo clique
4. O sistema irá gerar automaticamente um arquivo consolidado na mesma pasta

Simples assim ✅

---

## 📑 Ordenação dos Arquivos

A ordenação aplicada é:

> **Ordem alfabética crescente (A → Z)**  
> Baseada no nome do arquivo.

### 🔎 Exemplo:

Arquivos na pasta:


01 - Capa.pdf
02 - Índice.pdf
03 - Projeto.pdf
10 - Anexo.pdf


Serão unidos exatamente nessa sequência.

⚠️ Recomenda-se padronizar os nomes com numeração inicial (01, 02, 03...) para garantir a ordem desejada.

---

## 🎯 Quando usar?

- Muitos arquivos
- PDFs muito grandes
- Limitação de upload em plataformas online
- Processamento interno sem enviar documentos para a internet

---

## 🛠 Observações

- O PDF final será gerado com o nome: `PDF_FINAL_UNIDO.pdf`
- O arquivo será criado na mesma pasta onde o executável estiver
- Não é necessário instalar Python na máquina de destino
